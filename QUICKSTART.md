# Quick Start Guide

Get up and running with Image Converter in 3 simple steps!

## 1. Install Dependencies

```bash
npm install
```

## 2. Run the Application

```bash
npm run dev
```

The application will automatically launch in development mode with hot reload enabled.

## 3. Start Converting!

1. **Add Images**: Drag & drop images or click "Select Images"
2. **Choose Format**: Select output format (JPG, PNG, WebP, AVIF, or TIFF)
3. **Adjust Settings**: Set quality, resize options, metadata preferences
4. **Convert**: Click "Convert X Images" button
5. **Done!**: Converted images will be saved to your chosen output folder

## Common Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development mode |
| `npm run build` | Build for production |
| `npm run package` | Create test package |
| `npm run dist` | Create installers for distribution |

## Supported Formats

**Input**: JPG, JPEG, PNG, WebP, AVIF, TIFF
**Output**: JPG, PNG, WebP, AVIF, TIFF

## Default Settings

- **Quality**: 80%
- **Output Folder**: Same as source
- **Metadata**: Preserved
- **Resize**: Original size

## Tips

- Use quality 90-100 for high-quality images
- Use quality 60-80 for web-optimized images
- WebP and AVIF offer best compression
- PNG is best for images with transparency
- TIFF is best for professional/print work
